**Perceivable:**
- [ ] All images have descriptive alt text
- [ ] Color contrast ratios meet 4.5:1 minimum
- [ ] Text resizable up to 200% without loss of functionality
- [ ] Video content has captions/transcripts
- [ ] Audio descriptions for video content

**Operable:**
- [ ] All functionality keyboard accessible
- [ ] No keyboard traps
- [ ] Skip navigation links functional
- [ ] Focus indicators visible (especially on dark backgrounds)
- [ ] Sufficient time for form completion
- [ ] No flashing content (seizure risk)

**Understandable:**
- [ ] Clear, consistent navigation
- [ ] Form error messages helpful and specific
- [ ] Language declared in HTML
