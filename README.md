# Cioré™ | Official Website

This repository contains the full static website for Cioré, replicated from the Squarespace version.

## Project Overview
Cioré is creating sustainable smartphones that protect your privacy and respect the planet. This site serves as the primary landing page, information hub, and community contribution portal.

## Features
- **Privacy First OS Info**: Details about our custom operating system.
- **Sustainability Focus**: Information on solar-powered and eco-conscious hardware.
- **Waitlist Integration**: Join the movement and be the first to know.
- **Contribute & Donate**: Support the mission through genius or funding.

## Tech Stack
- HTML5
- CSS3 (Custom responsive design)
- JavaScript (Vanilla JS for interactivity)

## Deployment
This site can be hosted using GitHub Pages directly from this repository.
